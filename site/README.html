Qlam

Qlam is a lightwheight markup language designed to be easy to publishing,editing an modifing content of a static site web. Qlam is writtend in C++ and is under GNU-GPL licence. Syntax hilight files are available for Vim and Gedit

The cyle of the usage of Qlam is :

1. Edit text with a text editor like Vim, Gedit, Emacs ...

2. Get the HTML format page by qlam article.qlm

3. Upload the page on your site

Syntax

Comment

You can use html tags anaywhere.

In Qlam comment begins with # followed by a space.

	
# this is a comment

Produce <!-- this is a comment -->

Heading

#1 Title produce : <h1>Title</h1>

Paragraphs

Each paragraph begins on a new linge.
